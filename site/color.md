## ClassName

Use `.dark` to specify dark mode:

{% example %}
<div class="padding-3 background">
  <button type="button" class="button button--filled">Button</button>
</div>
<div class="dark padding-3 background">
  <button type="button" class="button button--filled">Button</button>
</div>
{% endexample %}
