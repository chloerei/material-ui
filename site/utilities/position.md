# Position

Format:

```
.position-{value}
```

Values:

- relative
- absolute
- fixed
- static
- sticky

Example:

{% example %}
<div class="position-relative background-primary-container" style="height: 60px;">
  <div class="position-absolute background-primary padding-1" style="bottom: 0;">
    Content
  </div>
</div>
{% endexample %}
